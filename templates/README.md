# 协议模板

本目录包含《智慧女性婚姻幸福的36堂法律必修课》书中引用的协议模板。

> **注意**：以下模板仅供学习参考。实际使用时，建议结合具体情况咨询专业律师。

## 模板列表

| 模板 | 用途 | 对应课程 |
|------|------|---------|
| 婚前财产协议 | 婚前约定双方财产归属 | 第3课、第23课 |
| 婚内财产协议 | 婚姻期间对财产进行约定 | 第23课 |
| 离婚协议 | 协议离婚时的综合约定 | 第30课 |

## 婚前财产协议核心注意事项

摘自本书第23课——婚前财产协议的9个要点：

1. 不要写"净身出户"等模糊表述
2. 不要写成"忠诚协议"
3. 不要写成"离婚协议"
4. 建议办理公证
5. 不要显失公平
6. 不要处分父母等第三人的财产
7. 协议涉及的财产类型要全面
8. 一式三份，一份放父母处或办公室
9. 每页签字按手印，双方签字后生效

## 离婚协议核心注意事项

摘自本书第30课——离婚协议的19个要点：

1. 写清楚对方不履行的违约责任
2. 约定对方不按时支付抚养费的违约金
3. 明确约定探望方式和违约责任
4. 不要写"精神赔偿"和"经济帮助"写反
5. 写清楚已知的债权和债务
6. 写清楚双方共同财产的分割方案
7. 房屋过户需要对方配合的，做委托公证授权自己办理
8. 对方名下的房子归自己，要先查询抵押或查封情况
9. 明确写清对方配合过户的时间及违约责任
10. 涉及对方还贷的，约定违约责任
11. 债务承担要约定清楚
12. 写清楚父母出资情况，尽可能让父母签协议
13. 不能处分第三人的财产
14. 检查关联协议内容
15. 对方住址和联系方式要准确
16. 约定让对方在规定期限内迁出户口
17. 约定让孩子在规定期限内办理转学或配合
18. 确认离婚协议的生效条件
19. 最终版本认真核对，防止内容被对方修改
