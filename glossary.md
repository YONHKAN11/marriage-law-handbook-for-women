# 婚姻家事法律术语表

> 本书涉及的核心法律概念简明解释。适用于读者快速查阅，也便于AI准确理解上下文。

## 婚姻基础

| 术语 | 解释 |
|------|------|
| **结婚登记** | 男女双方到婚姻登记机关办理结婚登记，领取结婚证后确立夫妻关系 |
| **事实婚姻** | 1994年2月1日前以夫妻名义同居，群众也认为是夫妻关系的，可认定为事实婚姻 |
| **无效婚姻** | 重婚、近亲结婚、未到法定婚龄等情形，婚姻自始无效 |
| **可撤销婚姻** | 因胁迫结婚或被隐瞒重大疾病的，可请求撤销婚姻 |
| **夫妻共同财产** | 婚姻关系存续期间所得工资、奖金、投资收益、知识产权收益、继承或受赠财产（遗嘱指定除外）等 |
| **个人财产** | 婚前财产、因身体伤害获得赔偿金、遗嘱或赠与指定归一方的财产、专用生活用品等 |

## 财产相关

| 术语 | 解释 |
|------|------|
| **婚前财产公证** | 婚前对双方财产进行公证，明确各自财产范围 |
| **婚内财产协议** | 夫妻在婚姻期间对财产归属作出的书面约定 |
| **夫妻共同债务** | 用于夫妻共同生活、共同生产经营的债务，或基于双方共同意思所负债务 |
| **个人债务** | 一方婚前债务、未经对方同意的个人举债、超出家庭日常需要的个人债务 |
| **彩礼** | 以缔结婚姻为目的，男方或男方家庭给予女方的财物 |
| **嫁妆** | 女方家庭为女儿结婚准备的财物，通常属于女方个人财产 |
| **房产共有** | 婚姻期间购买的房产，不论登记在哪方名下，一般属于夫妻共同财产 |
| **按份共有** | 房产证上明确标注各自份额 |
| **共同共有** | 房产证未标注份额，双方各占一半 |

## 离婚相关

| 术语 | 解释 |
|------|------|
| **协议离婚** | 双方就离婚、财产分割、子女抚养达成一致，到婚姻登记机关办理 |
| **诉讼离婚** | 一方不同意或无法协商一致时，向法院起诉离婚 |
| **离婚冷静期** | 协议离婚的30天冷静期，期满后30天内需双方共同申领离婚证 |
| **感情破裂** | 法院判决离婚的法定条件，包括重婚、与他人同居、家暴、遗弃、赌博吸毒恶习、感情不和分居满2年等 |
| **抚养权** | 离婚后谁直接抚养子女的权利 |
| **抚养费** | 不直接抚养子女一方支付的子女生活费、教育费、医疗费等 |
| **探望权** | 不直接抚养子女一方看望子女的权利 |
| **离婚损害赔偿** | 因一方重大过错导致离婚，无过错方可以请求损害赔偿 |
| **财产分割** | 离婚时对夫妻共同财产进行分配，一般均等分割，照顾子女、女方和无过错方 |
| **隐藏/转移财产** | 离婚时一方隐藏、转移、变卖、毁损夫妻共同财产或伪造债务，分割时对该方可以少分或不分 |

## 家庭暴力

| 术语 | 解释 |
|------|------|
| **家庭暴力** | 家庭成员间以殴打、捆绑、限制人身自由、经常性谩骂、恐吓等方式实施的身体、精神侵害 |
| **人身安全保护令** | 法院为保护家暴受害人签发，禁止施暴者实施家暴、骚扰、跟踪、接触 |
| **告诫书** | 公安机关对家暴加害人出具的法律文书，可作为家暴证据 |

## 法律程序

| 术语 | 解释 |
|------|------|
| **诉讼时效** | 民事权利的诉讼时效一般为3年，从知道或应当知道权利受损时计算 |
| **管辖法院** | 通常向被告住所地或经常居住地法院起诉；离婚案件也可由原告住所地法院管辖 |
| **举证责任** | "谁主张谁举证"，提出主张的一方需要提供证据 |
| **财产保全** | 为防止对方转移财产，可在诉讼前或诉讼中申请法院查封、冻结财产 |
| **强制执行** | 一方不履行生效判决或调解书时，另一方可申请法院强制执行 |
